# Titanic Survival Analysis

This project analyzes the Titanic passenger dataset using Python.

## Tools
- Python
- Pandas
- Matplotlib

## Analysis

The dataset is loaded directly from GitHub and basic analysis is performed.

Steps:
- Load dataset
- Inspect data
- Count survival distribution
- Visualize survival data

## Example Output

The script generates a bar chart showing how many passengers survived vs did not survive.

https://github.com/datasciencedojo/datasets/blob/master/titanic.csv
